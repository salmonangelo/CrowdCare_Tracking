Engineering Milestones

Roadmap

MVP → Pilot → Temple Deployment → Event Deployment → SaaS → Enterprise, with real dependencies at each step.

MVP
In progressEst. 3 weeks
Deliverables: stable risk engine, forecasting v1, alerting pipeline. Dependencies: GPU batching fix for multi-stream FPS, trend model on heatmap history.
Real-world Pilot
NextEst. 6–8 weeks after MVP
Deliverables: on-site fixed-camera install, live dashboard for one operator, incident log. Dependencies: a willing venue, on-site network/CCTV access, liability sign-off.
Temple Deployment
PlannedEst. Q4
Deliverables: queue-time estimation, festival-day stress testing. Dependencies: temple trust/administration approval, existing CCTV feed access.
Event Deployment
Planned
Deliverables: drone feed ingestion, dynamic zone detection for outdoor events. Dependencies: drone hardware + pilot license, GPS-zone mapping module.
SaaS Platform
Future
Deliverables: multi-tenant dashboard, self-serve onboarding, billing. Dependencies: 2+ successful pilots, hardened API auth, cloud infra.
Enterprise
Future
Deliverables: SLA-backed uptime, on-prem/edge options, compliance certifications. Dependencies: enterprise sales motion, security audits.
